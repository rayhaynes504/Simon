SIMON

About Simon

Simon is an electronic game of memory skill invented by Ralph H. Baer and Howard J. Morrison, working for toy design firm Marvin Glass and Associates, with software programming by Lenny Cope. This is my rendition of their game.

How to play

Press the start button to begin the game. The game will display a sequence of flashing lights the user has to repeat. Once the user has repeated the sequence, click the "Compare" button to see if the sequence was repeated accurately. If the user successfully repeats the sequence, the game will generate a tougher (longer) sequence for the user to repeat. Press the start button again to display the new sequence. This process is repeated until the user incorrectly repeats a sequence, then the game is over.
